# stickchart

### how to use

`git clone git@github.com:boyko-pavlo/stickchart.git && yarn && yarn build`

`yarn install`

`yarn build`

`cd stickchart`

`cd ..`

`git clone git@github.com:metaferryman/pari-market-web-trader.git`

`cd pari-market-web-trader`

`git checkout chart`

`yarn install`

`yarn run autolink`

`yarn run build:dev`
